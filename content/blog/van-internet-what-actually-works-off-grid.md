Reliable van internet comes from layered redundancy, not one magic device.

## Baseline Setup

- Primary hotspot with external antenna support
- Secondary carrier or tether backup
- Mounting and cable routing that protects signal quality

## Common Mistakes

- Chasing theoretical max speeds
- Using only one network
- Ignoring power draw of network hardware
